# VanessaLoansHouston.com

Interactive mockup + site spec for Vanessa Martinez's lender/agent-recruiting website.

- `index.html` — clickable HTML/CSS/JS prototype (bright/sunny theme, parallax homepage, AI chat demo, calculators)
- `VanessaLoansHouston_Site_Spec.docx` — full site specification

This is a design mockup, not the production site. No real loan applications are processed here.
